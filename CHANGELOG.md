# Status Nerd Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Set a status on Slack, GitLab and GitHub at once, with per-service selection
- Turn a few notes into AI-generated status suggestions (requires Raycast Pro), shuffle through them, then set
- **Meeting Status**: generate a privacy-guarded status from your current or next calendar meeting
- **Select Status**: pick from recent, saved and default statuses, with an emoji picker for your own
- **Random Status** and **Clear Status** one-click commands
- Configurable status duration and a first-run tone onboarding
